#  Healthcare Analytics: Predicting Hospital Readmissions

##  Overview

This project predicts whether a diabetic patient will be readmitted within 30 days using machine learning.

##  Dataset

* Source: UCI Machine Learning Repository
* Records: ~100,000 hospital visits

##  Approach

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA) with visualizations
* Logistic Regression model

##  Key Insights

* Dataset is imbalanced
* Age and hospital stay duration impact readmission
* Model performs better on non-readmitted cases

##  Outcome

Helps identify high-risk patients so hospitals can take preventive action.

##  Tech Stack

Python, Pandas, Scikit-learn, Matplotlib, Seaborn
